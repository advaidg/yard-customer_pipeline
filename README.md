## Quick Start

```bash
# 1. Copy env file and fill in agent URLs
cp .env.example .env

# 2. Build the orchestrator image
docker build -t yard-customer_pipeline:latest .

# 3. Run
docker run --env-file .env -p 9000:9000 yard-customer_pipeline:latest

# 4. Invoke
curl -X POST http://localhost:9000/invoke \
  -H "Content-Type: application/json" \
  -d '{"input": {"message": "hello"}}'

# 5. Health check
curl http://localhost:9000/health
```
